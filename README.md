# A Museum of My English Education

A compact 3D interactive museum for the English reflection assignment.

The experience frames the course texts as a four-year progression:

- Grade 9: identity, family, belonging, and dreams
- Grade 10: fear, survival, ambition, and collapse
- Grade 11: society, control, ideology, and systems
- Grade 12: mortality, isolation, memory, responsibility, and uncertainty
- Final Room: the player becomes the final exhibit

Each book now has its own side room with three clickable exhibit objects. Clicking a glowing doorway opens a short room preview, and clicking objects inside a room opens symbolism/theme analysis.

## Run

Serve the folder locally, then open the local URL in a browser.

```powershell
npm run dev
```

Then visit:

```text
http://localhost:5173
```

Controls:

- W/S or Arrow Up/Down: move through the museum
- A/D or Arrow Left/Right: turn
- Mouse drag: look around
- Mouse wheel: move forward/back
- Number keys 1-6: jump to a specific wing for presentation/testing
